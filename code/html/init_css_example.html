引用

引用分为两种，一种是 pre 标签引用，另一种是 blockquote 标签引用。

这是带有 bg-color-blue 类的 pre 标签引用。
根据 pre 标签的性质，当代码中文字换行时，展示内容也会换行；当代码中有多个连续的空格（例如：        ）时，展示内容也会展示多个空格。
而且，当宽度不够时，会出现横向滚动条，而不会自动换行。
